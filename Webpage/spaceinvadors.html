SPACE INVADORS

ABOUT THE GAME:

How to play

The controls for Space invadors are the Left and Right keys for moving the player across the screen, and Space key for shooting the bullet. Toggle Numpad 0 for switching 'ON' or 'OFF' the autoplay

Objective

The objective of this game is to kill the enemies bu shooting them with a bullet from your space ship. You can move your space ship horizontally across the screen. The game end if you fail to kill the enemy in time and they crash into you. The enemies keeps spawing and hence it is a continuous game of taking your aim for the moving targets and shooting.

Player

The player is defaulted in the centre of the screen. The value of change in the x-axis of the player is defaulted to zero but are accessed though the left and right keys. The function player blits a preloaded player image onto the screen using the blit method defined in the draw class in pygame framework.

Enemy

The position of enemies to spawn is selected at random using the random module but is defined to a specific area and direction for movement. Each time the enemy hits the boundary, it drops down a level coming closer to you. Now a for loop is used for definig random values to the list of enemies. And there after an function blits an image loaded into the program using the load method in image class.

enemyImg = []

enemyX = []

Autoplay

The auto play function checks x-axis and direction of the lowest enemy and aims towards it based on its y-axis.

def selfaim(enemyX, enemyY, playerX, enemyX_change):

 maxY = max(enemyY)

 ind = enemyY.index(maxY)

 maxX = enemyX[ind]

 dir = enemyX_change[ind]

 try:

  d = round((480 - maxY) / (playerX - maxX), 2)

 except ZeroDivisionError:

  return 0

 if dir == 4:

  if d > 2.40:

   return 5

  elif d < 2.20:

   return -5

  else:

   return 0

 else:

  if d > -2.20:

   return 5

  elif d < -2.40:

   return -5

  else:

   return 0
