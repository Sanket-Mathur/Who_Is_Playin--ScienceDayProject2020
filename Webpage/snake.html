SNAKE

ABOUT THE GAME:

How to play

The game snake is played using the arrow keys. The arrow keys define the direction in which the snake will be going. Use Numpad 0 to turn 'ON' or 'OFF' the autoplay.

Objective

The objective of this game is to eat the food and stay alive as long as possible. If you crash into the wall or yourself, the snake will die however the games become more and more chalanging as the game proceds since each block of food increases the length of the snake by one.

Crashing

Checking if the snake has crashed is made by creating a function which takes the list of player position as its parameter. It checks if the head of the snake is overlapping over its body or if it is out of the screen bounds. It returns True in case the snake has crashed and returns False if not.

def crash(pos):

 l = [pos[x] for x in range(1, len(pos))]

 if pos[0] in l:

  return True

 elif (pos[0][0] >= 620) or (pos[0][0] < 20) or (pos[0][1] >= 620) or (pos[0][1] < 20):

  return True

 else:

  return False
