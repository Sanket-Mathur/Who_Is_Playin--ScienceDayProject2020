JUMP AND TUCK

ABOUT THE GAME:

How to play

The game jump and tuck is controlled using the upper and lower arrow keys. Use Numpad 0 to turn 'ON' or 'OFF' the autoplay.

Objective

The objective of this game is to dodge the obstacles coming in the way of the skater. You have two choices of either jumping over the obstacle or tucking under it. Deciede it based on the height on the obstacle.

CODE SNIPTS:

Player

The player has a predefined x and y axis which can change based upon the position of the player. The image of the player in the bent mode and in the standing mode in loaded into the memory using the load method in the image class and is blitted over the screen using the blit method defined in the pygame framework.

playerimg_stand = pygame.image.load('skate.png')

Autoplay

The auto play function checks if the obstacle is withing the limits of the player and which type of obstacle is it. Depending on that it passes its commands to either jump or tuck in order to dodge the obstacle.

def auto(obsX, o):

 if o == 1 and obsX >= 0 and obsX <= 300:

  return 'tuck'

 elif o == 2 and obsX >= 0 and obsX <= 200:

  return 'jump'

 else:

  return 'none'
